## 📝 Название и описание проекта 
**Insight Search** — интеллектуальная система мониторинга и семантического анализа пользовательских мнений в социальной сети Instagram.

Программа позволяет брендам переходить от простого поиска слов к глубокому пониманию контекста. Система автоматически классифицирует отзывы по темам (Цена, Качество, Сервис), определяет тональность с учетом сарказма и рассчитывает приоритетность реагирования для SMM-специалистов.

## 🛠 Технические требования
* **Язык:** Python 3.10+
* **Менеджер пакетов:** pip
* **Зависимости:** streamlit, transformers, torch, pymorphy3, pandas, plotly, wordcloud.

## 🚀 Основные возможности
* **Hybrid Sentiment Analysis:** Комбинация нейросети **RuBERT-tiny** и экспертных словарей-триггеров.
* **Thematic Classification:** Группировка комментариев по категориям (Цена, Логистика, Качество, Поддержка).
* **Priority Scoring:** Алгоритм ранжирования важности сообщений от 1.0 до 5.0.
* **Interactive Dashboard:** Визуализация аналитики, динамики обсуждений и облака смыслов.
