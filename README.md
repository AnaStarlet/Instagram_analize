## 📝 Обзор проекта
Проект представляет собой интеллектуальную систему мониторинга и анализа пользовательских мнений в социальной сети Instagram. Система переходит от простого поиска слов к семантическому анализу текста, позволяя брендам оперативно выявлять репутационные угрозы и потребительские тренды.

### Основные возможности:
*   **Hybrid Sentiment Analysis:** Определение тональности (Позитив, Негатив, Нейтрально) с помощью дообученной нейросетевой модели **RuBERT-tiny** и экспертных словарей-триггеров.
*   **Priority Scoring:** Алгоритм автоматического ранжирования сообщений по степени важности (от 1.0 до 5.0) на основе эмоциональной окраски, длины текста и критических ключевых слов.
*   **Thematic Classification:** Группировка комментариев по категориям: *Цена, Логистика, Качество, Поддержка*.
*   **Interactive Dashboard:** Визуализация данных (облако слов, графики Plotly, динамика обсуждений).

## 🛠 Технологический стек
*   **Язык:** Python 3.10+
*   **NLP:** `transformers` (Hugging Face), `pymorphy3` (морфологический анализ).
*   **ML Модель:** `cointegrated/rubert-tiny-sentiment-balanced`
*   **Интерфейс:** Streamlit (Web-Dashboard).
*   **База данных:** SQLite (локальное хранилище).
*   **Аналитика:** Pandas, Plotly, WordCloud.
