Название проекта
Insight Search — это интеллектуальная система для мониторинга и анализа пользовательских мнений в социальной сети Instagram.
Программа выполняет автоматическую загрузку комментариев, определяет их тональность (Hybrid Sentiment Analysis) с помощью нейросети RuBERT-tiny, классифицирует по темам (Цена, Качество и др.) и рассчитывает индекс приоритетности для SMM-специалистов. Результаты визуализируются в виде интерактивного дашборда.
Требования
Язык: Python 3.10+
Менеджер пакетов: pip
Основные библиотеки: streamlit, transformers, torch, pymorphy3, pandas, plotly, wordcloud.
Обзор возможностей
Hybrid Sentiment Analysis: Сочетание нейросетевой модели и экспертных словарей-триггеров для точного определения эмоций и сарказма.
Priority Scoring: Алгоритм ранжирования важности сообщений (от 1.0 до 5.0) на основе лингвистических признаков.
Thematic Classification: Автоматическая группировка отзывов по категориям.
Interactive Dashboard: Наглядная визуализация статистики и облака смыслов в реальном времени.
